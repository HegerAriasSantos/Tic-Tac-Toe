# Tic-Tac-Toe

A simple tic-tac-toe game. Made for practicing concepts such as:

- React components.
- State Management.

[Check the game rules.](https://en.wikipedia.org/wiki/Tic-tac-toe)

## Start the Project

### In Production Mode

```
	# Install the required dependencies
	$ npm install

	# Build the project
	$ npm build
```

### In Development Mode

```
	# Install the required dependencies
	$ npm install

	# Start the project
	$ npm dev
```
